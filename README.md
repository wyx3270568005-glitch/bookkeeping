# bookkeeping

`feishu-bookkeeping` 是一个用于自动记账的 Agent Skill。你只要在任意Agent的聊天窗口发送记账消息，Agent会自动调用飞书CLI在指定的多维表格《账本》内写入记录。

它最初由作者在Hermes Agent上随手创建，只依靠一个skill.md也能跑起来。作者的Hermes安装在绿联云DX4600的docker上。作者把Hermes绑定到了飞书和微信等平台作为聊天机器人，通过聊天窗口发送自然语言、结构化字段或带金额的图片时，Agent基于skill.md命中为记账请求后，就能完成记账。在使用过程中，用户可以随时纠正agent，让skill不断进化，做到了真正的“越用越流畅，越用越懂你”。（不是

## 如何安装
安装`skill骨架`＞安装`飞书CLI`＞绑定一个飞书`《账本》`＞开始记账
### 一句话安装
把下面这段话直接粘贴给你的Agent：
```text
帮我把bookkeeping skill安装到本地，你需要先把这个代码仓库完成地下载到本地，保存在当前agent的skill根目录。由于每台设备的依赖和网络环境不同，请你自行选择可用的下载命令和镜像源。注意，先完整克隆整个仓库，不要遗漏任何目录和文件。完成克隆后，把该skill的安装路径和目录树及目录树解释发给用户。然后，阅读skill.md和readme.md，自动推进流程，仅在必要时引导用户手动介入。直到Ta完成第一笔记账才视为完全安装成功。https://github.com/andy-JustSayWhen/bookkeeping.git
```

把下面这段话直接粘贴给你的Agent：

**Step2：一段话，让Agent帮你安装Skill的依赖**
这个记账skill主要依赖飞书CLI完成账本的读写删等功能，如果你此前已经安装并配置了飞书CLI，那么无需重复安装。跳过此步骤，直接看Step3。

把下面这段话直接粘贴给你的Agent：

```text
帮我安装飞书CLI，严格按照官方文档的四个步骤。完成安装和验证。注意，第2至4步都有链接生成，需要用户点击完成验证。请你把相关链接显示在聊天窗口内显示，提醒用户点击它们完成验证并告诉你已完成，你继续推进流程。整个安装完成后，你需要自行调取飞书CLI完成一次测试，测试内容：向用户返回飞书云文档库的目录树，包含云盘、知识库、我的文档库三个路径在内，每个路径下最多展示3个文档名，其余用 ... 省略。同时，把你首次安装和调用遇到的问题及优化方案保存到skill.md的<## agent调用飞书CLI可能遇到的问题及解决办法>。

飞书CLI官方安装引导文档：https://open.feishu.cn/document/no_class/mcp-archive/feishu-cli-installation-guide.md
```
**Step3：一段话，让Agent绑定飞书账本**

情况1：如你已有《账本》且已保存在飞书，把下面这段话直接粘贴给你的Agent：
```text
我在飞书已有账本，你帮我找到它，然后按照skill.md的<## 记录账本信息>记录和更新账本基本信息。如果你不确定某个或某几个文档是否为我的账本，请发送带序号的名称清单给我，我选择一个序号进行绑定。或者，让我把账本名称或链接发你，你读取后绑定。一切准备就绪后，提醒可以开始记账及支持的记账触发方式。
```

情况2：如你已有《账本》但未保存在飞书，把下面这段话直接粘贴给你的Agent：
```text
我在飞书没有账本，我会向你发送一个格式为csv/excel的账本文件，你帮我上传为飞书多维表格，命名为《账本》，保存到我的文档库路径下。然后按照skill.md的<## 记录账本信息>记录和更新账本基本信息。一切准备就绪后，提醒可以开始记账及支持的记账触发方式。
```

情况3：如你在飞书没有《账本》，把下面这段话直接粘贴给你的Agent：
```text
我在飞书没有账本，你帮我创建一个名为《账本》的多维表格，默认字段包括：名称、金额、类型、账本、日期。然后按照skill.md的<## 记录账本信息>记录和更新账本基本信息。一切准备就绪后，提醒可以开始记账及支持的记账触发方式。
```
**Step4：开始记账**

从现在开始，你可以通过三种方式触发记账：
- 自然语言。例如：`记一笔，东方树叶 5 元`
- 结构化字段。例如：`早餐 20 支出 个人`
- 发送图片。例如：发送微信、飞书、支付宝、订单、账单等含金额图片



## 如何使用
### 记账方式
用户可以在 Agent 的聊天窗口中通过以下方式记账：

- 自然语言。例如：`记一笔，东方树叶 5 元`
- 结构化字段。例如：`早餐 20 支出 个人`
- 发送图片。例如：发送微信、飞书、支付宝、订单、账单等含金额图片
- 其他。用户根据自己的需求，自定义记账方式，告诉Agent今后你想怎么记账即可。Agent会新增触发条件。

### 自我进化
- Agent会从《账本》生成<关键词➡️字段映射表>，你的记录越多，Agent的推断能力就越强。用不了多久，Agent只要在图片或文字中拿到`名称`和`金额`字段，其他字段就都可以智能推断并完成填充；
- Agent对于<关键词➡️字段映射表>无法推断的字段，会调用当前大模型能力进行智能推断。
- 任何时候，你都可以通过自然语言，让Agent新增记账方式和规则。
