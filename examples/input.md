# 输入示例

这些示例用于帮助 Agent 学习用户的记账表达习惯。处理细节以根目录 [SKILL.md](../SKILL.md) 为准。

## 自然语言支出

```text
记一笔，东方树叶 5 元
```

```text
刚才打车 12.5
```

```text
午饭花了 28
```

```text
充了 30 话费
```

## 工作账本支出

```text
小米微博投流 300
```

```text
制作费 500，工作账本
```

```text
采购键盘 199 工作
```

## 收入

```text
收到项目款 1200，工作账本
```

```text
报销到账 68.5
```

```text
退款 18 元
```

## 结构化字段

```text
早餐 20 支出 个人
```

```text
打车 12.5 支出
```

```text
项目款 1200 收入 工作
```

## 图片或截图

```text
用户发送一张微信、飞书、支付宝、账单或订单截图，图片内包含金额。
```

处理要求：先识别图片文字和金额；检测到金额后直接按记账流程执行，不追问“要不要记账”。

## 明确取消

```text
这张图不是记账
```

处理要求：取消写入。

## 修改 Skill 规则

```text
以后“茶叶蛋”归到个人账本。
```

```text
把“采买”识别为工作账本。
```

```text
退回 CHG-20260429-151552 前版本
```
