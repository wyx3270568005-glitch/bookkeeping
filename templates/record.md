# 单笔记账模板

用于生成写入后的单笔记录回复。字段值应来自飞书写入前后的最终记录。

## 字段

- 名称：`{{name}}`
- 金额：`{{amount}}`
- 类型：`{{type}}`
- 账本：`{{account_book}}`
- 日期：`{{date}}`
- 项目归属：`{{project}}`
- 标签：`{{tag}}`
- 账本链接：`{{ledger_url}}`

## 回复格式

```text
记账成功！

       名称            金额     类型    账本    项目归属     标签
─────────────────────────────────────────────────────────────────
     {{name}}       {{amount}}  {{type}}  {{account_book}}  {{project}}  {{tag}}

账本链接：{{ledger_url}}
```

## 填充规则

- 金额保留两位小数。
- 类型只能是 `支出` 或 `收入`。
- 账本优先使用用户指定值；未指定时按 `SKILL.md` 规则推断。
- 日期默认使用记录创建时的北京时间；用户明确指定日期时按用户指定。
- 项目归属和标签用于回复展示时，必须来自已配置的字段或当前 Skill 规则；没有可用规则时使用 `无` 或 `其他`。
- 不使用 Markdown 表格；使用代码块保持等宽对齐。
